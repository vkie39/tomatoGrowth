# Installation
## Requirements
- Python 3.10+ (tested on 3.10.5, 3.11.4)
- Windows 10, macOS (tested on both OS)
## Install Libraries
```bash
pip install -r requirements.txt
```

## Get yolov5 repository
```bash
git clone https://github.com/ultralytics/yolov5
```
